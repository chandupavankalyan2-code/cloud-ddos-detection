# Dataset Download Instructions

The dataset is too large to store in this repository.

**Download from:** https://zenodo.org/records/14681803

After downloading, place the file here as:
```
data/AIS_Dataset_on_Cloud_DDoS_Attacks.csv
```

**Dataset details:**
- 52,318 network flow records
- 80 features (flow-based, packet-based, statistical)
- Binary labels: BENIGN / DDOS
- Near-balanced class distribution
